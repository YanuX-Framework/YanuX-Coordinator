@yanux/coordinator

YanuX Coordinator

This is part of the YanuX Framework. It is a TypeScript library that allows developers to connect their application code to the YanuX Framework, namely the YanuX Auth and YanuX Broker components.

It abstracts the communication the YanuX Broker API and provides tools for the automatic distribution of UI components based on device capabilities, and their manual redistribution, as well as the management and sharing of running application states.
